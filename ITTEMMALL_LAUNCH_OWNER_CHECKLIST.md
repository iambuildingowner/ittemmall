# 잇템몰 런칭/PG 심사 오너 체크리스트

잇템몰은 기존 사업자의 대표 쇼핑몰이자 Toss PG/네이버페이 심사용 허브로 운영한다. Codex는 화면, 정책 페이지, 주문 흐름, 설정 자리, 검증 스크립트를 준비하고, 아래 실제 값은 오너가 확인 후 제공한다.

## 현재 상태

- 로컬 작업본: 잇템몰 헤더/푸터/정책/상품 구조 반영 완료
- 상품: 잇템몰 안에 MARCE 피클볼 패들 2종 등록 완료
- 사업자정보: 마르세 스포츠 기준 기존 사업자 정보 차용 완료
- 도메인 연결: `https://ittemmall.com` 접속 가능, `www`도 대표 도메인으로 이동 확인
- 라이브 반영: `https://ittemmall.com`에 잇템몰 작업본 반영 완료
- 라이브 보호: 내부 문서, 예시 설정, private/scripts/output 경로 403 차단 확인
- SEO 기본값: `robots.txt`, `sitemap.xml`, canonical, 구조화 데이터 반영 확인
- 배포 기준: GitHub Actions가 `build/ittemmall-public` 공개 묶음만 Hostinger public web root로 업로드
- 통신판매업 신고번호: 잇템몰 기준 신고 또는 변경신고 후 반영 필요
- 현재 남은 서버 게이트: 운영 서버 private config, 주문 저장 JSON 경로, 관리자 토큰, PG/Naver Pay 발급 키 입력 필요

## 진행 순서

1. 잇템몰 작업본을 `https://ittemmall.com` 라이브에 반영한다. 완료
2. 라이브 공개 파일과 내부 파일 노출 여부를 확인한다. 완료
3. 구매안전서비스 이용확인증을 먼저 발급받는다.
4. 정부24에서 통신판매업 신규신고 또는 변경신고를 잇템몰 URL 기준으로 진행한다.
5. 통신판매업 신고번호가 나오면 잇템몰 하단/정책/약관에 반영한다.
6. 토스 PG 심사와 MID/API 키 발급을 진행한다.
7. 운영 서버 private config와 주문 저장소를 설정한다.
8. 토스 결제 확인 후 네이버페이 결제형 신청을 진행한다.

- 이번 프로젝트 우선순위: 잇템몰 라이브 준비 후 토스 또는 은행 에스크로 경로로 확인증 확보

## 현재 남은 일 분리

### 대표님이 직접 해야 하는 일

1. 구매안전서비스 이용확인증 발급 경로 선택
   - 우선 후보: 토스페이먼츠 통신판매업 바로신청 또는 은행 에스크로
   - 필요 정보: 사업자등록번호, 대표자 인증, 사업자 계좌/기업뱅킹 인증, 잇템몰 URL
2. 정부24 통신판매업 신고 또는 변경신고 제출
   - 필요 정보: 대표자 인증, 사업자 정보, 도메인 `https://ittemmall.com`, 호스트서버 소재지, 구매안전서비스 이용확인증
3. 등록면허세 납부 및 신고증/신고번호 확인
4. 토스페이먼츠 전자결제 신청
   - 필요 정보: 사업자 정보, 정산 계좌, 담당자 정보, 잇템몰 URL, 판매 상품/가격/배송·환불 정책
5. 토스 심사 승인 후 MID, client key, secret key 전달
6. 네이버페이 결제형 신청
   - 필요 정보: 실명인증된 네이버 아이디, 사업자 서류, PG 거래/연동 증빙, 담당자 이메일, 판매 상품/금액/배송 정보

### Codex가 처리할 일

1. 구매안전서비스 확인증 발급 화면에서 입력값 점검
2. 정부24 신고 입력값 정리 및 제출 전 사이트 하단/정책 재확인
3. 신고번호 발급 후 잇템몰 하단, 사업자정보 페이지, 약관/정책에 신고번호 반영
4. 토스 PG 심사 보완 요청 대응용 화면/문구/정책/주문 흐름 수정
5. Toss MID/client key/secret key 수령 후 public/private 설정 분리 반영
6. 운영 서버 private config와 주문 저장소 경로 설정 보조
7. 네이버페이 clientId/chainId/clientSecret 수령 후 public/private 설정 분리 반영
8. 네이버페이 검수 체크리스트, return/approve/cancel 흐름, 운영 URL 스모크 테스트 보조

## 공식 기준 확인 메모

- 정부24 민원 안내는 선지급식 통신판매의 경우 구매안전서비스 이용확인증을 제출 서류로 안내한다.
- 찾기쉬운 생활법령정보는 통신판매업 신고 시 도메인, 호스트서버 소재지, 구매안전서비스 확인증 또는 결제대금예치 이용확인증을 함께 본다고 안내한다.
- 토스페이먼츠 자료는 신고 전 사업자등록증, 홈페이지, 구매안전서비스 이용확인증을 먼저 준비하라고 안내한다.
- 토스페이먼츠 PG 심사 자료는 상품 1개 이상, 실제 상품금액, 배송/환불 정책, 하단 사업자정보, 통신판매업 신고번호 등을 주요 점검 항목으로 본다.
- 네이버페이 결제형 자료는 서류 심사 후 인증정보 전달, 개발/테스트, 검수 체크리스트, 서비스 환경 검수, 결제로직 취약성 검수 순서로 진행된다고 안내한다.
- 네이버페이 결제형 심사 서류는 공통 거래금액 증빙 또는 매출 발생 내역, 개인사업자 사업자등록증 사본, 법인사업자 추가 법인 서류 등을 요구한다.
