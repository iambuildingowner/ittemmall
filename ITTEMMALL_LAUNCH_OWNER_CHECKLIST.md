# 잇템몰 런칭/PG 심사 오너 체크리스트

잇템몰은 기존 사업자의 대표 쇼핑몰이자 Toss PG 심사용 허브로 운영한다. 네이버페이 가맹점 신청/연동은 별도 마르세 스포츠 프로젝트에서 진행한다. Codex는 화면, 정책 페이지, 주문 흐름, 설정 자리, 검증 스크립트를 준비하고, 아래 실제 값은 오너가 확인 후 제공한다.

실제 제출 화면을 켜놓고 진행할 때는 `ITTEMMALL_OWNER_GATE_ACTION_BOARD_2026-06-10.md`를 실행 체크리스트로 사용한다.

## 진행 순서

1. 잇템몰 작업본을 `https://ittemmall.com` 라이브에 반영한다. 완료
2. 라이브 공개 파일과 내부 파일 노출 여부를 확인한다. 완료
3. 구매안전서비스 이용확인증을 먼저 잇템몰 URL 기준으로 발급 또는 변경 발급받는다.
4. 정부24에서 기존 통신판매업 변경신고를 잇템몰 URL 기준으로 진행한다.
5. 변경 완료된 신고번호/신고정보를 잇템몰 하단/정책/약관에 반영한다. 신고번호가 유지되더라도 공정위/신고증의 도메인이 잇템몰 기준으로 바뀐 뒤 반영한다.
6. 토스 PG 심사와 MID/API 키 발급을 진행한다.
7. 운영 서버 private config와 주문 저장소를 설정한다.
8. 잇템몰 본몰에서는 네이버페이 신청을 진행하지 않는다. 네이버페이는 마르세 스포츠 프로젝트로 분리한다.

## 공식 기준 확인 메모

- 정부24 민원 안내는 선지급식 통신판매의 경우 구매안전서비스 이용확인증을 제출 서류로 안내한다.
- 찾기쉬운 생활법령정보는 통신판매업 신고 시 도메인, 호스트서버 소재지, 구매안전서비스 확인증 또는 결제대금예치 이용확인증을 함께 본다고 안내하고, 신고사항 변경 시 변경신고서와 변경사항 증빙서류를 제출한다고 안내한다.
- 토스페이먼츠 자료는 신고 전 사업자등록증, 홈페이지, 구매안전서비스 이용확인증을 먼저 준비하라고 안내한다.
- 토스페이먼츠 PG 심사 자료는 상품 1개 이상, 실제 상품금액, 배송/환불 정책, 하단 사업자정보, 통신판매업 신고번호 등을 주요 점검 항목으로 본다.
- 네이버페이 공식 기준 확인은 마르세 스포츠 별도 스레드에서 다시 수행한다.
