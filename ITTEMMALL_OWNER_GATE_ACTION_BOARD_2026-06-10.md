## 1. 구매안전서비스 이용확인증

목표: 기존 통신판매업을 잇템몰 기준으로 변경신고하기 전에, `https://ittemmall.com` 기준 구매안전서비스 이용확인증을 확보한다.

먼저 확인할 질문:

| 체크 | 질문 | 답 |
|---|---|---|
| [ ] | 기존 통신판매업 신고 때 사용한 구매안전서비스 확인증 발급처가 어디인가? |  |
| [ ] | 기존 확인증에 표시된 쇼핑몰명/URL은 무엇인가? | 현재 공정위 도메인은 `https://www.byuljangdaejang.com/` |
| [ ] | 해당 발급처에서 URL을 잇템몰로 변경하거나 새 확인증을 받을 수 있는가? |  |

핵심 판단:

- 대표님은 이미 통신판매업 신고가 있으므로 신규신고가 아니라 변경신고로 간다.
- 토스 `통신판매업 바로신청`은 신규 신고까지 묶이는 흐름일 수 있으므로 그대로 끝까지 진행하지 않는다.
- 토스 화면에서 `신규 통신판매업 신고 제출`, `링크페이 쇼핑몰 생성`, `새 통신판매업 신고` 흐름이 나오면 멈춘다.
- 지금 필요한 것은 신고 제출이 아니라 `잇템몰 기준 구매안전서비스 이용확인증`이다.

발급 경로 선택:

| 선택 | 경로 | 메모 |
|---|---|---|
| [ ] | 기존 구매안전서비스 발급처 | 기존 통신판매업 신고에 이미 붙어 있는 발급처가 있으면 잇템몰 URL로 변경/재발급 가능한지 먼저 확인 |
| [ ] | 은행 에스크로 | 기업/사업자 계좌가 있는 은행에서 잇템몰 기준 확인증 발급 가능 여부 확인 |
| [ ] | 토스페이먼츠 PG/상점관리자 | 토스 PG 계약 또는 상점관리자에서 확인증만 발급/출력 가능한 경우 사용 |
| [ ] | 토스 통신판매업 바로신청 | 확인증만 받을 수 있는 화면인지 확인 후 진행. 신규신고 제출 흐름이면 중단 |

입력값:

|---|---|
| 쇼핑몰명 | 잇템몰 |
| URL | https://ittemmall.com |
| 상호 | 공간 |

| 사업자등록번호 | 221-31-17043 |

발급 후 Codex에 넘길 것:

| 체크 | 항목 |
|---|---|

## 공식 기준 메모

- 정부24/생활법령 안내 기준으로, 통신판매업 신고에는 구매안전서비스 이용확인증이 필요하고 신고사항 변경 시에는 변경사항 증빙서류를 제출한다.
- 토스페이먼츠 전자결제 심사 안내는 상품 준비, 하단 사업자정보, 통신판매업 신고번호, 결제 연동을 주요 심사 항목으로 본다.
- 네이버페이 가맹점 가이드는 결제형을 가입 신청/심사, 개발 연동, 운영 단계로 나눠 안내한다.
- 대표님은 기존 통신판매업 신고가 있으므로 이 프로젝트의 행정 단계는 신규신고가 아니라 변경신고다.
