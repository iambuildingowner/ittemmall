# White Web Design Theme

잇템몰 숨김 단독몰이나 상품 상세를 `화이트 테마`로 만들 때 적용하는 기본 디자인 기준이다.

## 핵심 기준

- 전체 배경, 섹션 배경, 강조 카드 배경은 기본적으로 흰색을 유지한다.
- 큰 면적을 검정, 진한 남색, 진한 포인트 컬러로 채우지 않는다.
- 강조는 색을 많이 칠하는 방식이 아니라 얇은 테두리, 구분선, 여백, 작은 포인트 컬러로 만든다.
- 포인트 컬러는 버튼, 작은 라벨, 선, 아이콘, 선택 상태 정도에 제한한다.
- 검정 배경 카드, 진한 색 배경 박스, 무거운 대비 블록은 화이트 테마에서 기본 금지한다.

## 상세페이지 섹션

- 핵심 포인트 박스도 흰색 배경을 유지하고 얇은 테두리만 둔다.
- 포인트 목록은 굵은 컬러 배경 대신 얇은 왼쪽 선, 작은 번호, 간결한 문장으로 정리한다.
- 상품 이미지와 카피가 이미 분위기를 만들도록 두고, UI 박스가 시선을 빼앗지 않게 한다.
- 배경을 채워야 하는 경우에도 연한 회색이나 아주 옅은 포인트 컬러까지만 사용한다.

## 작업 시작 질문

신사업발굴 또는 잇템몰 상품 게시를 시작할 때 사용자가 테마를 말하지 않았다면 먼저 묻는다.

```text
이번 숨김 단독몰/상세페이지는 어떤 테마로 갈까요?

1. 화이트 테마
2. 블랙 테마
3. 포인트 컬러 테마
4. 기존 잇템몰 본점 톤 유지
```

사용자가 별도 답을 주지 않으면 상품군과 타깃에 맞춰 Codex가 추천하되, 추천 이유를 짧게 말하고 진행한다.
