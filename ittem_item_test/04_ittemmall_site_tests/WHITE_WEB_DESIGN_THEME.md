# White Web Design Theme

잇템몰 본사이트, 숨김 단독몰, 상품 상세를 만들 때 별도 지시가 없으면 적용하는 기본 디자인 템플릿이다.

## 핵심 기준

- 전체 배경, 섹션 배경, 강조 카드 배경은 기본적으로 흰색을 유지한다.
- 큰 면적을 검정, 진한 남색, 진한 포인트 컬러로 채우지 않는다.
- 기본 템플릿 자체는 흰색과 검정 중심으로 만든다.
- 강조는 색을 많이 칠하는 방식이 아니라 얇은 테두리, 구분선, 여백으로 만든다.
- 하늘색, 민트, 핑크 같은 포인트 컬러는 기본 템플릿에 넣지 않는다. 사용자가 특정 상품 전용 포인트 컬러를 지정할 때만 해당 전용몰 범위에서 사용한다.
- 검정 배경 카드, 진한 색 배경 박스, 무거운 대비 블록은 화이트 테마에서 기본 금지한다.

## 상세페이지 섹션

- 핵심 포인트 박스도 흰색 배경을 유지하고 얇은 테두리만 둔다.
- 포인트 목록은 굵은 컬러 배경 대신 얇은 왼쪽 선, 작은 번호, 간결한 문장으로 정리한다.
- 상품 이미지와 카피가 이미 분위기를 만들도록 두고, UI 박스가 시선을 빼앗지 않게 한다.
- 배경을 채워야 하는 경우에도 연한 회색까지만 사용한다.

## 상품 카드 썸네일

- 상품 카드 썸네일은 `1080 x 1350px`, 비율 `4:5`를 기본값으로 강하게 고정한다.
- 카드 클래스는 `product-card is-portrait`를 기본으로 본다.
- 썸네일 비율은 `4 / 5`를 따른다.
- 이미지 태그는 `width: 100%`, `height: 100%`, `object-fit: contain`을 기본으로 둔다.
- 상품 원본 이미지 비율이 달라도 카드 썸네일 영역의 규격은 흔들리지 않게 한다.
- 가로 이미지나 정사각형 이미지를 그대로 카드 대표 이미지로 쓰지 않는다. 카드용 `1080 x 1350px` 파일을 별도로 만들어 연결한다.

## 상품 이미지 생성

- 상품 상세페이지용 대표 이미지와 상세 이미지를 생성할 때, 상품 카드용 세로형 썸네일도 함께 만든다.
- 썸네일 이미지는 반드시 `1080 x 1350px`, 비율 `4:5` 기준 세로형으로 만든다.
- 썸네일은 카드에서 상품이 작아 보이지 않도록 여백을 과하게 두지 않는다.
- 메인 상세 이미지, 광고 이미지, 카드 썸네일은 같은 제품 톤과 촬영 분위기를 유지한다.

## 작업 시작 기준

신사업발굴 또는 잇템몰 상품 게시를 시작할 때 사용자가 별도 테마를 말하지 않으면 이 화이트 기본 템플릿으로 시작한다.

사용자가 특정 상품 전용 포인트 컬러를 지정하면, 그 색은 해당 숨김 단독몰 안에서만 제한적으로 사용한다.
