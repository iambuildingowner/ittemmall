# 잇템몰 광고 이미지 참고 소재 선정 기준

목적: 잇템몰 아이템 검증 광고를 만들 때 타사 이미지를 베끼지 않고, 네이버 쇼핑과 Meta 광고 라이브러리에서 소비자가 바로 이해하는 이미지 구조만 참고한다.

## 기본 원칙

- 잇템몰 광고 이미지 생성 요청이 들어오면 이 문서의 절차를 먼저 수행한다.
- 이미지 생성 도구를 먼저 실행하지 않는다.
- 광고 이미지는 타깃 키워드를 기준으로 찾는다.
- 네이버 쇼핑과 Meta 광고 라이브러리를 함께 본다.
- 타사 이미지는 구도, 강조 방식, 정보 배치만 참고한다.
- 원본 이미지, 로고, 문구, 브랜드 요소, 제품 형태를 그대로 복제하지 않는다.
- 최종 광고 이미지의 한글 문구와 서체는 별도 가이드가 없으면 `Pretendard`를 기본으로 쓴다.
- 최종 광고 이미지는 기존 상세페이지 이미지를 조합해서 배너처럼 만들지 않는다.
- 네이버 쇼핑과 Meta 광고 라이브러리 후보 중 최종 기준이 될 예시 이미지 1개를 고른다.
- 최종 광고 이미지는 그 예시 이미지 1개의 구도, 톤, 정보량을 기준으로 새롭게 생성한다.
- 우리 상세페이지 이미지는 제품 일관성 확인용 참고로만 본다.
- 사용자가 준 타깃 키워드는 광고 소재의 출발점이다. 상품명, 광고 이미지 안 짧은 카피, Meta 광고글 첫 문장에 그 키워드를 자연스럽게 포함한다.
- 키워드는 소비자가 한눈에 이해하도록 쓰되, 키워드 반복/검색어 나열/어색한 직역투로 만들지 않는다.
- 잇템몰 광고 이미지는 가능하면 `사진 생성 후 별도 텍스트 오버레이`로 만들지 않는다. 최종 이미지 생성 프롬프트 안에 짧은 카피와 가격을 함께 넣고, 이미지와 자연스럽게 어울리는 최종 광고 이미지로 한 번에 생성한다.
- 작업 순서는 `1차 제품/착용 이미지 생성` 후 `2차 Meta 광고 업로드용 최종 이미지 생성`으로 둘 수 있다. 2차 프롬프트에는 카피라이팅과 금액을 직접 넣고, `이미지와 잘 어울리게 넣어라`, `글씨체는 Pretendard로 해달라`처럼 짧고 명확하게 지시한다.
- 사진형 광고 이미지 생성 프롬프트에는 항상 `AI 느낌이 나지 않게`, `실제와 같은 하이 퀄리티 사진`을 짧게 포함한다.
- 이미지 안 텍스트는 최소화한다. 기본은 짧은 카피 1개와 가격 정도만 넣는다.
- 이미지 안 한글 문구는 중국어/영어 상세페이지를 그대로 번역한 듯한 직역투, 어색한 띄어쓰기, 기계번역식 표현을 금지한다. 한국 쇼핑 광고에서 자연스럽게 보이는 짧은 표현으로 쓴다.
- 이미지 안에 도형, 박스, 배지, 칩을 만들고 그 안에 글자를 넣을 때는 기본적으로 가로/세로 가운데 정렬한다. 의도적으로 좌측 정렬하는 디자인을 선택할 때만 예외로 둔다.
- 기능 라벨, 설명 박스, 보조컷, 상세페이지식 정보 나열을 과하게 넣지 않는다.
- 생성 결과가 이상하면 문제 섹션만 잘라 붙이는 방식으로 고치지 않는다. 같은 예시 이미지 기준으로 전체 이미지를 다시 생성한다.

## 네이버 쇼핑 참고 기준

1. 타깃 키워드를 네이버 쇼핑에 검색한다.
2. 상위 10개 상품 후보를 본다.
3. 브랜드 상품은 기본 배제하고, 비브랜드 상품을 1순위로 본다.
4. 비브랜드 후보 중 소비자 입장에서 키워드와 가장 직관적으로 연결되는 이미지를 고른다.
5. 비브랜드 상품이 거의 없거나 전부 브랜드 상품이면 브랜드 이미지를 참고할 수 있다.
6. 참고할 때는 제품 배치, 착용 형태, 기능이 보이는 위치, 보조컷 구성을 본다.

## Meta 광고 라이브러리 참고 기준

1. 대한민국, 모든 광고 기준으로 타깃 키워드를 검색한다.
2. 상단 노출 광고를 우선 본다.
3. 정지 이미지 광고 소재를 우선 참고한다.
4. 동영상 프레임 캡처, 영상 썸네일, 광고 카드 전체 캡처는 기본 참고 이미지로 보지 않는다.
5. 정지 이미지 광고가 부족할 때만 영상 소재를 보조 참고한다.
6. 2주 이상 게재된 광고는 지속성 신호로 별도 기록한다.

## 최종 소재 선택 순서

1. 네이버 쇼핑 상위 10개 중 비브랜드 직관 이미지 후보를 고른다.
2. Meta 광고 라이브러리에서 정지 이미지 광고 후보를 고른다.
3. 둘 중 소비자가 타깃 키워드를 가장 빨리 이해할 수 있는 예시 이미지 1개를 선택한다.
4. 선택한 예시 이미지 1개의 구도, 톤, 정보량을 기준으로 새 광고 이미지를 생성한다.
5. 새 광고 이미지를 기존 광고에 교체하거나 같은 광고세트 안에 새 광고로 추가해 비교한다.

## 금지되는 진행 방식

- 네이버 쇼핑과 Meta 광고 라이브러리 확인 없이 바로 광고 이미지를 생성하지 않는다.
- 동영상 내부 프레임 캡처를 정지 이미지 광고 참고로 취급하지 않는다.
- 광고 카드 전체 캡처를 광고 이미지 참고로 취급하지 않는다.
- 타사 정지 이미지를 그대로 업로드하거나, 로고/문구/제품 형태를 복제하지 않는다.
- 기존 상세페이지 이미지를 단순 조합해 광고 배너처럼 만들지 않는다.
- 여러 참고 이미지를 섞어 복잡한 합성 방향으로 가지 않는다. 최종 기준은 예시 이미지 1개다.
- 문제 섹션만 잘라 새로 붙이는 부분 패치 방식으로 수정하지 않는다.

## 선풍기조끼 적용 예시

- 타깃 키워드: `선풍기조끼`
- 네이버 쇼핑 기준: 상위 10개 중 비브랜드 팬 조끼 이미지 우선. 팬 위치, 착용 형태, 앞뒤 구조가 바로 보이는 이미지가 우선이다.
- Meta 라이브러리 기준: 정지 이미지 광고에서 제품 대형컷, 팬/쿨링 강조, 앞/뒤 보조컷, 짧은 후킹 문구 배치를 참고한다.
- 최종 기준: 네이버 쇼핑 또는 Meta 라이브러리 후보 중 소비자에게 가장 직관적인 예시 이미지 1개를 고른다.
- 최종 생성 방향: 예시 이미지의 구도/톤/정보량을 기준으로 새 이미지를 생성한다.
- 이미지 안 문구: 짧은 카피 1개와 가격 정도만 넣는다. 사용자가 준 키워드가 `선풍기조끼`라면 카피에도 자연스럽게 반영한다. 예: `등까지 시원한 선풍기 조끼`, `48,900원`
